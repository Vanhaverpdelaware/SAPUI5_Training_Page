IDE Setup

This page will guide you through setting up your development environment for SAPUI5 and TypeScript development. You can use either VS Code or SAP Business Application Studio (BAS).

Note: If you already have VS Code installed with the SAP Fiori Tools extension, you can skip to Node.js & Tooling.

Install VS Code

Download and install VS Code from the delaware Company Portal. Search for Visual Studio Code in the portal app.
