Node.js & UI5 Tooling

Node.js

SAPUI5 Tooling requires Node.js 20.x LTS. Verify your installation:

node --version   # Should output v20.x.x
npm --version    # Should output 10.x.x or higher

If not installed, download from the Company Portal or from nodejs.org (choose the LTS version).

Version matters: Node.js 21 is not supported by UI5 Tooling v4. Stick to v20 LTS or v22 LTS.

UI5 CLI

Install the UI5 Tooling CLI globally — this is the main tool for serving and building UI5 apps:

npm install --global @ui5/cli

Verify the installation:

ui5 --version

easy-ui5 Generator

The easy-ui5 generator scaffolds new SAPUI5 projects in seconds. Install it once globally:

npm install -g yo generator-easy-ui5

Create a new TypeScript UI5 project:

yo easy-ui5 project

Follow the prompts. Select TypeScript when asked for the project type.

Git

Git 2.x+ is required. Verify:

git --version

Configure your identity if not already done:

git config --global user.name "Your Name"
git config --global user.email "your.name@delaware.pro"

CF CLI (for BTP deployment)

Required for deploying to SAP BTP Cloud Foundry. Install CF CLI 8.x+:

cf --version   # verify after install

Download from the CF CLI GitHub releases page.

Quick checklist: Run all four version commands above. If all output a version number, you are ready to start.
