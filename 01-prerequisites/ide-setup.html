IDE Setup

VS Code (Recommended)

VS Code is the recommended IDE for SAPUI5/TypeScript development at delaware. Install it from the Company Portal on your Windows machine.

  1. Open the Company Portal app (search in Start menu)
  2. Search for Visual Studio Code
  3. Click Install

Required VS Code Extensions

Install these extensions from the VS Code Extensions panel (Ctrl+Shift+X):

Extension ID Purpose
SAP Fiori Tools sapse.sap-ux-fiori-tools-extension-pack UI5 development, scaffolding
TypeScript Next ms-vscode.vscode-typescript-next TypeScript support
ESLint dbaeumer.vscode-eslint Code linting
Prettier esbenp.prettier-vscode Code formatting
XML Tools DotJoshJohnson.xml XML view formatting

SAP Business Application Studio (BAS)

BAS is SAP's cloud IDE — an alternative to VS Code that comes pre-configured with all SAP tooling. No local installation needed, but requires a BTP subaccount.

When to use BAS: Prefer BAS for BTP-connected projects or when working without a local development machine. Use VS Code locally for faster iteration and offline work.

Recommended VS Code Settings

Add this to your project's .vscode/settings.json:

{
  "editor.formatOnSave": true,
  "editor.defaultFormatter": "esbenp.prettier-vscode",
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": "explicit"
  },
  "typescript.tsdk": "node_modules/typescript/lib"
}
