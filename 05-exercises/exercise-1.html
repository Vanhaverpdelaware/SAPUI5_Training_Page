Exercise 1

In this exercise you scaffold a new SAPUI5 TypeScript project from scratch and get it running locally. No prior UI5 knowledge assumed — just follow the steps.

Prerequisites: Complete the Node.js & UI5 Tooling setup page before starting.

Goal

By the end of this exercise you will have a running SAPUI5 TypeScript app displaying a list of items from a JSON model, with a working sidebar navigation and a detail view.

Step 1 — Scaffold the project

# Create a new folder and open it
mkdir ui5-training-app
cd ui5-training-app

# Scaffold with easy-ui5
yo easy-ui5 project

When prompted:

  • Project name: ui5.training.app
  • Namespace: ui5.training
  • UI5 version: 1.120.0
  • TypeScript: Yes
  • Add Karma tests: No (for now)

Step 2 — Install dependencies

npm install

Step 3 — Start the dev server

ui5 serve --open index.html

The browser opens at http://localhost:8080. You should see the scaffolded app. Open the browser console (F12) — there should be no errors.

Step 4 — Explore the project structure

Locate and open these files in VS Code:

  • webapp/manifest.json — app descriptor, routing config
  • webapp/view/Main.view.xml — the main XML view
  • webapp/controller/Main.controller.ts — the controller
  • ui5.yaml — tooling config
  • tsconfig.json — TypeScript config

Step 5 — Add a JSON model

In Main.controller.ts, add a JSON model in onInit():

import Controller from "sap/ui/core/mvc/Controller";
import JSONModel from "sap/ui/model/json/JSONModel";

/** @namespace ui5.training.controller */
export default class Main extends Controller {
  public onInit(): void {
    const oModel = new JSONModel({
      items: [
        { id: "1", name: "Material A", quantity: 100 },
        { id: "2", name: "Material B", quantity: 250 },
        { id: "3", name: "Material C", quantity: 50 }
      ]
    });
    this.getView()?.setModel(oModel);
  }
}

Step 6 — Display the list in the view

In Main.view.xml, replace the placeholder content with a list:

<mvc:View controllerName="ui5.training.controller.Main"
  xmlns="sap.m" xmlns:mvc="sap.ui.core.mvc">
  <Page title="Materials">
    <content>
      <List items="{/items}">
        <StandardListItem
          title="{name}"
          description="Qty: {quantity}" />
      </List>
    </content>
  </Page>
</mvc:View>

Save the file — the browser auto-refreshes and shows your 3 items.

Step 7 — Verify TypeScript is working

Introduce a deliberate type error in the controller:

// Add this line in onInit() — it should show a red underline
const x: number = "this is wrong";

VS Code should show a TypeScript error. Remove the line after confirming it works.

Done! You now have a working SAPUI5 TypeScript app with a JSON model bound to a list. Continue to Exercise 2 to add routing and a detail view.
